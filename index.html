linux-command-line

经过几个周的思沃大讲堂之旅，学到了许多新的知识。特别是以前只是略有接触的前端技术，在几次作业的训练之后有了进一步的了解，实践是最重要的，只有抛给自己一个任务动手完成才能有所收获，这是这几周最大的感受。其次反思自己，一开始学习命令行时很不适应也没有分配buddy，所以关于这方面的知识经过后来内容的迭加，逐渐被我疏忽了。每次应用的时候都仅限于自己的舒适区，越是熟悉的命令使用越多，越不熟悉的反而越少，意识到这是个很不好的事情，我决定在看书这件事情上选择从
命令行 看起，下面是最初的一些笔记，一方面整理出来方便回顾，另一方面借此练习markdown的语法。

linux-command-line

第二章：什么是shell

  1. 上下键：历史命令; 左右键：移动光标
  2. 复制粘贴：用鼠标拖动选中一段或双击选中单词，按中建。只能复制terminal中的内容
  3. 查看当前在那个版本的shell：echo $SHELL
  4. $是shell提示符，出现时表示用户可以输入; 如果是#，则表示有超级用户(管理员)的权限
  5. bash 是来自 GNU 项目的 shell 程序，“bash” 是 “Bourne Again SHell” 的首字母缩写， 指bash 是最初 Unix 上由 Steve Bourne 写成 shell 程序 sh 的增强版。
  6. date : 显示当前系统时间; cal : 显示当前月份的日历
  7. df : 查看磁盘剩余空间的数量; free : 显示空闲内存的数量
  8. exit : 结束终端会话
  9. ctrl+alt+f1…f6：打开虚拟终端，需要输入用户名和密码
  10. alt+f7：退出虚拟终端，返回图形界面（注意在自己机子上这个快捷键不起作用，要用ctrl+alt+f8）

第三章：文件系统中跳转

  1. 显示当前工作目录：pwd(print working directory),默认初始工作目录是/home/me
  2. 当以 普通用户身份操控系统时，home是为唯一允许用户对文件进行写入的地方。
  3. 列出当前目录内容：ls
  4. 更改当前目录：cd+路径
  5. 绝对路径：开始于根目录，以/开头
  6. 相对路径：开始于当前工作目录，利用 . 或者 .. 跳转，前者表示当前目录，后者表示上级目录pwd
  7. cd的快捷键：
    • cd : 到家目录
    • cd - : 到先前目录
    • cd ～userame : 到家目录
  8. 关于文件名的规则：
    • 以 . 开头的文件是隐藏文件，需要用ls -a 才能看到
    • 文件名和命令大小写敏感
    • linux没有文件扩展名，但是有些应用程序会有。
    • 文件名可包含空格 . - _ ,但是不要在文件名中使用空格，可以用下划线代替。

第四章：探究操作系统

  1. ls : 列出当前目录中内容
    • ls /user : 列出指定目录的内容，ls后加指定目录
    • ls ~ /user：列出多个指定目录的内容，ls后加指定目录，空格分开
    • ls -l：结果以长模式输出
  2. 命令的构成：命令名+一个或多个用来更正命令行为的选项+表示命令对象的一个或多个参数 ;
    command -options arguments
  3. 大多数命令使用的选项由一个中划线加上一个字符组成，例如，“-l”，但是许多命令也支持长选项，长选项由两个中划线加上一个字组成。 许多命令也允许把多个短选项串在一起使用。如ls -lt这个例子，ls 命令有两个选项， “l” 选项产生长格式输出，“t”选项按文件修改时间的先后来排序。ls -lt –reverse 加上长选项 “–reverse”，则结果会以相反的顺序输出。
